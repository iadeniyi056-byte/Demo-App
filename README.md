# Ade Bank 💰

A beautiful demo banking web application built with HTML, CSS & JavaScript

 ✨ Features

- Modern Login Page
- Interactive Dashboard with balance
- Virtual Debit Card
- Working Transfer Modal
- Recent Transactions
- Bottom Navigation
- Hide/Show Balance

🛠️ Technologies Used

- HTML5
- CSS3 (Bootstrap 5)
- Vanilla JavaScript
- LocalStorage
